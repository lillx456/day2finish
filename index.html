요약: 웹서버에 정보를 저장해 놓은 집합체.


본문 웹 브라우저를 통해 볼 수 있는 웹 콘텐츠(텍스트, 이미지, 오디오, 비디오 등)와 링크나 동적 기능으로 호출되는 내외부 리소스를 포함한 관련 웹 페이지들의 집합체.

일반적으로 ‘www’로 시작하는 도메인 이름으로 식별되고, 하나 이상의 웹 서버에서 호스팅된다. 웹 사이트는 기본적으로 HTML, XHTML 등의 언어를 사용하여 제작되며, 이미지, 오디오, 비디오 등의 멀티미디어 콘텐츠를 포함할 수 있다.

또 웹 서버의 동적 기능을 통해 데이터베이스, 파일, 응용 프로그램 등의 외부 리소스를 호출해 보여 준다. 기본적으로 HTTP 프로토콜을 통해 접속 가능하며, HTTPS를 이용하면 암호화를 통해 웹 콘텐츠의 보안과 개인 정보 보호가 가능하다.

최초의 웹 사이트는 1990년 팀 버너스 리(Tim Berners-Lee)가 유럽입자물리학연구소(CERN)에서 만든 ‘info.cern.ch’이다. 이후 인터넷 기술의 발전과 함께 수많은 웹 사이트가 생겨났고, 2016년 기준으로 전 세계 10억 개 이상의 웹 사이트가 존재하고 있다. 웹 사이트는 PC뿐만 아니라, TV, 스마트폰 등 인터넷에 연결된 다양한 디바이스를 통해 이용할 수 있다.
